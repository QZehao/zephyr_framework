# 脚本与工具说明

本页说明仓库根目录 **`scripts/`** 下脚本的用途与典型用法，便于新成员快速上手。**不构成**对第三方工具（Zephyr SDK、West、OpenOCD 等）的完整教程。

**相关**：[环境搭建与配置指南.md](../10-环境与构建/11-环境搭建与配置指南.md) · [版本管理.md](../70-发布与产品化/71-版本管理.md)

---

## 1. 环境变量（构建前）

| 脚本 | 平台 | 说明 |
|------|------|------|
| **`setup_env.ps1`** | Windows PowerShell | 从 **`zephyr_config.env`** 读取并设置 **`ZEPHYR_BASE`**、**`ZEPHYR_SDK_INSTALL_DIR`** 等（若实现中包含） |
| **`setup_env.bat`** | Windows CMD | 同上，批处理版本 |
| **`setup_env.sh`** | Linux / macOS | `source scripts/setup_env.sh` |

使用前请已准备好 **`zephyr_config.env`**（由 **`zephyr_config.env.template`** 复制）。

---

## 2. 版本与发布

| 脚本 | 说明 |
|------|------|
| **`bump_version.py`** | 统一提升 **`APP_VERSION`** 及 README/Doxyfile 等中的版本字符串（用法见 **`python scripts/bump_version.py -h`** 或仓库说明） |
| **`package_release.ps1` / `package_release.sh`** | 打包发布产物（若项目启用；以脚本内注释为准） |

详见 **[版本管理.md](../70-发布与产品化/71-版本管理.md)**、**[发布检查清单.md](../70-发布与产品化/73-发布检查清单.md)**。

---

## 3. 文档（API）

| 脚本 | 说明 |
|------|------|
| **`generate_docs.ps1`** | Windows：调用 **Doxygen** 生成 **`docs/api/html`** |
| **`generate_docs.sh`** | Linux / macOS：同上 |

需本机已安装 **Doxygen**（及可选 **Graphviz**）。生成后打开 **`docs/api/html/index.html`**。

---

## 4. 构建辅助

| 脚本 | 说明 |
|------|------|
| **`build_all.bat` / `build_all.sh`** | 批量构建多个目标（若仓库中已配置具体板卡列表；以脚本内容为准） |
| **`analyze_map.ps1` / `.bat` / `.sh`** | 分析链接 **`zephyr.map`**，辅助查看内存占用（用法见脚本内注释） |

---

## 5. 未随仓库提供的工具

部分文档示例中可能出现 **串口监视**、**一键烧录** 等命令，若 **`scripts/`** 下无对应文件，请直接使用：

- **`west flash`** / **`west debug`**（见 **[烧录与调试快速指南.md](61-烧录与调试快速指南.md)**）  
- 主机侧 **pyserial miniterm**、**PuTTY** 等通用工具

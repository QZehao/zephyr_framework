| 文件 | 用途 | 是否提交到 Git |
|------|------|----------------|
| `prj.conf` | Zephyr 配置（`CONFIG_*`） | ✅ 是 |
| `Kconfig` | 配置菜单定义 | ✅ 是 |
| `app.overlay` | 设备树覆盖（通用） | ✅ 是 |
| `zephyr_config.env` | 本地路径配置 | ❌ 否（由 template 复制） |
| `APP_VERSION` | 应用版本号 | ✅ 是 |

## 🆘 常见缩写

| 缩写 | 全称 | 含义 |
|------|------|------|
| **RTOS** | Real-Time Operating System | 实时操作系统 |
| **MCU** | Microcontroller Unit | 微控制器 |
| **SoC** | System on Chip | 片上系统 |
| **GPIO** | General Purpose Input/Output | 通用输入输出口 |
| **UART** | Universal Asynchronous Receiver/Transmitter | 通用异步收发传输器 |
| **SPI** | Serial Peripheral Interface | 串行外设接口 |
| **I2C** | Inter-Integrated Circuit | 集成电路总线 |
| **RAM** | Random Access Memory | 随机存取存储器 |
| **ROM** | Read-Only Memory | 只读存储器 |
| **OTA** | Over-The-Air | 空中升级（固件更新） |
| **NVS** | Non-Volatile Storage | 非易失性存储 |
| **IPC** | Inter-Process Communication | 进程间通信 |
| **ISR** | Interrupt Service Routine | 中断服务程序 |
| **DTS** | Devicetree Source | 设备树源码 |
| **DTB** | Devicetree Blob | 编译后的设备树 |
| **YAML** | YAML Ain't Markup Language | 配置文件格式（如 `.github/workflows/ci.yml`） |
